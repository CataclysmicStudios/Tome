/// @slug spawn-enemy-usage
Usage Example:
This method allows you to quickly populate the area around the EntitySpawner's initial coordinates.
```gml 
// Initialize the spawner at x: 250, y: 300
var _cave_spawner = new EntitySpawner(250, 300);

// Spawn a standard mob of enemies
_cave_spawner.spawn_enemy("goblin", 5);

// Spawn a single, tougher enemy type
_cave_spawner.spawn_enemy("orc_brute", 1);
```
